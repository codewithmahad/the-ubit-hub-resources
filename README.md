# The UBIT Hub Resources

A curated collection of notes, solved assignments, lecture questions, exam preparation material, and academic resources compiled for UBIT students.

The purpose of this repository is simple:

* Preserve useful course material.
* Reduce repeated requests for notes and resources.
* Help current and future students prepare more effectively.
* Build a long-term academic archive for UBIT.
